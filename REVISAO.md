### 1. Criar repositório localmente

```bash
git init
```

### 2. Criando uma nova branch (o nome da branch criada é 'main')

```bash
git checkout -b main
```

ou

```bash
git branch -M main
```

### 3. Adicionar alterações para a área de preparação

```bash
git add nome_do_arquivo
```

ou

```bash
git add .
```
